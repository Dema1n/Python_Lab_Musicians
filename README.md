# Musicians_Parser

Этот проект предназначен для парсинга музыкальных групп, их альбомов, песен из альбомов, а также текстов этих песен с использованием Genius API и хранения данных в базе данных.

## Основные функции
- Интеграция с Genius API для получения текстов песен
- Парсинг информации о музыкальных группах и треках
- Сохранение данных в БД

## Структура проекта
- main.py — основной скрипт приложения
- config.py — API токен
- requirements.txt — зависимости проекта
- data/ — директория для данных

## Установка и запуск
1. Клонируйте репозиторий
2. Установите зависимости: pip install -r requirements.txt
3. Запустите: python main.py
